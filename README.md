# 📖 Overview

Event Project Manager is a desktop application designed to simplify the process of organizing events and managing project-related activities.

The application provides a centralized environment for handling:

- Event Planning
- Project Management
- Task Organization
- Participant Management
- Workflow Tracking
- Operational Monitoring

The system focuses on usability, simplicity, and efficient management of project activities.

# 🚀 Running the Application

## Method 1: Run from Visual Studio (Recommended)

1. Clone the repository

```bash
git clone https://github.com/Kerolosnady1/Project_Manager.git
```

2. Open:

```text
Event_Project_Demo-KN.slnx
```

3. Build the solution

4. Run the application

```text
F5
```

or

```text
Start Debugging
```

---

## Method 2: Run Executable Directly

Navigate to:

```text
Event_Project_Demo-KN/bin/Debug/net10.0-windows/
```

Run:

```text
P2P.exe
```

No additional setup is required.
